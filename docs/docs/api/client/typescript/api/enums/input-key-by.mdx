# InputKeyBy <s generated />

These are the methods you choose to run keystrokes.

#### UNITY_KEYCODE <s generated />

- The type that is used for keyboard input in projects with [Unity Input Manager](https://docs.unity3d.com/Manual/class-InputManager.html) .

#### UNITY_KEYBOARD <s generated />

- The type used by projects with [Unity Input System](https://docs.unity3d.com/Packages/com.unity.inputsystem@latest) to enter the keyboard.
