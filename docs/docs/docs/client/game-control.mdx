## Key input

User keystrokes can be simulated through `Gamium Client`.
[KeyBy](../../api/client/typescript/classes/key-by) allows you to specify the key input method.

You can use it as below.

<Tabs groupId="language" defaultValue="ts">
<TabItem value="ts" label="TypeScript" default>

```ts
await gamium.sendKey(KeyBy.unityKeycode('Space'));
await gamium.sendKey(KeyBy.unityKeyboard('Space'));
```

</TabItem>

<TabItem value="py" label="Python" default>

```python
gamium.send_key(KeyBy.unity_keycode('Space'));
gamium.end_key(KeyBy.unity_keyboard('Space'));
```

</TabItem>
</Tabs>

## ActionChain

When manipulating the game through the `Gamium Client`, if you want to perform multiple actions continuously, you can use the [`Action Chain`](../../api/client/typescript/classes/action-chain) to perform the action at the most accurate time.

You can add multiple actions to [`ActionChain`](../../api/client/typescript/classes/action-chain). When you call the performance() of [`ActionChain`](../../api/client/typescript/classes/action-chain), the added actions are performed sequentially.

You can combine several actions as shown below.

#### Usage

<Tabs groupId="language" defaultValue="ts">
<TabItem value="ts" label="TypeScript" default>

```ts
await gamium
  .actions()
  .move({ x: screen.width / 2, y: screen.height / 2 })
  .click({ x: screen.width / 2, y: screen.height / 2 })
  .perform();

await gamium
  .actions()
  .sendKeys([KeyBy.unityKeycode('W'), KeyBy.unityKeycode('A')])
  .sendKeys([KeyBy.unityKeycode('Space')])
  .perform();
```

</TabItem>

<TabItem value="py" label="Python" default>

```python
gamium.actions()
  .move(Vector2(screen.width / 2, screen.height / 2 ))
  .click(Vector2(screen.width / 2, screen.height / 2 ))
  .perform();

gamium.actions()
  .send_keys([KeyBy.unity_keycode('W'), KeyBy.unity_keycode('A')])
  .send_keys([KeyBy.unity_keycode('Space')])
  .perform();
```

</TabItem>
</Tabs>
