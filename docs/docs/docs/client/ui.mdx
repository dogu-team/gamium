You can obtain an object called [`UI`](../../api/client/typescript/classes/ui) through the [`Game Driver`](../../api/client/typescript/classes/gamium-client).
[`UI`](../../api/client/typescript/classes/ui) provides an interface for easy UI manipulation.

You can use it as below.

##### Usage

<Tabs groupId="language" defaultValue="ts">
<TabItem value="ts" label="TypeScript" default>

```ts
const ui = gamium.ui();
await ui.click(By.path('Object path'));
```

</TabItem>
<TabItem value="py" label="Python" default>

```python
ui = gamium.ui()
ui.click(By.path('Object path'))
```

</TabItem>
</Tabs>

### Auto wait

If you are using the class [`UI`](../../api/client/typescript/classes/ui), automatically do the [`wait`](../../api/client/typescript/classes/ui-element) until you find it.
The above Usage works the same as below.

<Tabs groupId="language" defaultValue="ts">
<TabItem value="ts" label="TypeScript" default>

```ts
const button = await ui.wait(By.path('Object path'));
await gamium.wait(Until.elementInteractable(button));
await button.click();
```

</TabItem>
<TabItem value="py" label="Python" default>

```python
button = ui.wait(By.path('Object path'))
gamium.wait(Until.element_interactable(button))
button.click();
```

</TabItem>
</Tabs>
