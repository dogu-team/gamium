## Setting up your local environment

### Step 1: Fork

Fork the project on GitHub and clone your fork locally.

```sh
$ git clone https://github.com/dogu-team/gamium.git
$ cd gamium
$ git remote add upstream https://github.com/dogu-team/gamium.git
$ git fetch upstream
```
