# GamiumClient <s generated />

Provides the ability to interact with the `Gamium Engine`. Gamium Client can communicate with game builds with `Gamium Engine SDK`.
The `Gamium Client` sends the specified message and the `Gamium Engine SDK` that receives the message performs the logic corresponding to the message.

The `Gamium Client` can be used as follows.

##### Usage

```python
from gamium import *

service = TcpGamiumService("127.0.0.1", 50061)
gamium = GamiumClient(service)
gamium.connect()
```
