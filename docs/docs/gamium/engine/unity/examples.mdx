In unity, [Input Manager](https://docs.unity3d.com/Manual/class-InputManager.html) and [Input System](https://docs.unity3d.com/Packages/com.unity.inputsystem@latest) can be used for the development of Input control,
[UGUI](https://docs.unity3d.com/2021.3/Documentation/Manual/com.unity.ugui.html) or [UI Toolkit](https://docs.unity3d.com/2021.3/Documentation/Manual/UIElements.html) is available for UI development.

In line with these various cases, we have prepared a project containing gamium in advance.
You can find it in the github repository below.


- [gamium-engine-unity-samples](https://github.com/dogu-team/gamium-engine-unity-samples)


It is difficult to respond to various cases, but gamium team is working hard to support all of them.
