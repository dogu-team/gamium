# GamiumClient <s generated />

`Gamium Engine`와 상호작용하는 기능을 제공합니다. GamiumClient는 `Gamium Engine SDK`가 탑재된 게임빌드와 통신할 수 있습니다. 
`Gamium Client`는 지정된 메시지를 보내고 해당 메시지를 수신받은 `Gamium Engine SDK`가 메시지에 해당되는 로직을 수행합니다. 

`Gamium Client`는 아래와 같이 사용할 수 있습니다.

##### Usage

```ts
import { prepareGamium } from '@dogu-tech/toolkit';

const gamiumKit = await prepareGamium();
const gamium = gamiumKit.gamium;
```
