`Object path`는 `Gamium Client`가 사용해 원하는 Object를 조회하고 조작할 때 사용됩니다.

:::info
Object Path는 아래 방법을 이용하여 찾을 수 있습니다.

- [`Console Inspector`](/host-and-device/device/streaming-and-remote-control/game-ui-inspector)
- `Unity` - [`Gamium Editor`](../gamium-engine/unity/query-objects)

:::

<br />

아래는 `Gamium Client`에서 `Object path`를 사용하는 예시입니다.

```ts
const cameraObj = await gamium.findObject(By.path('/Main Camera[1]'));
```

위 예시에서는 `'/Main Camera[1]'`라는 `Object path`를 이용하여 camera를 찾고 있습니다. [`By.path()`](api/classes/by.mdx) 함수를 사용하는 이유는 여러 Object 검색 방법중에 `Object path`를 통해 검색함을 알리기 위함입니다.

<br />

다른 방법으로는 아래와 같이 tag를 이용하여 찾을 수도 있습니다.

```ts
const cameraObj = await gamium.findObject(By.tag('MainCamera'));
```
