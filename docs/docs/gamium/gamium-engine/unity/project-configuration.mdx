`Gamium`을 사용하기 위해선 `Gamium Engine SDK`를 프로젝트에 추가해야합니다.
`Gamium Engine SDK`는 `Gamium Client`가 요청한 기능을 수행하여 자동화된 테스트를 진행 할 수 있도록 도와줍니다.

## 1단계 : Gamium Engine SDK 추가하기

[Project manifest](https://docs.unity3d.com/Manual/upm-manifestPrj.html)애서 `dependencies`에 다음과 같이 추가합니다.

```
"com.dogu.gamium.engine.unity": "https://github.com/dogu-team/gamium-engine-unity.git#1.0.0",
```

:::info
Gamium Engine SDK 소스코드와 예시프로젝트는 아래 링크에서 확인할 수 있습니다

- [gamium-engine-unity](https://github.com/dogu-team/gamium-engine-unity)
- [gamium-engine-unity-samples](https://github.com/dogu-team/gamium-engine-unity-samples)

:::

## 2단계 : Scripting Define Symbol 추가

`Gamium`이 동작하기 위해선 **`USE_GAMIUM`** 라는 `Scripting Define Symbol`이 필요합니다. **`USE_GAMIUM`**을 설정하여야 `Gamium Engine`가 정상 실행됩니다.  
[Custom scripting symbols](https://docs.unity3d.com/Manual/CustomScriptingSymbols.html) 가이드를 참고하여 `Script Compilation` 항목으로 이동 후 **`USE_GAMIUM`** 를 추가합니다.

:::tip
`Gamium Engine SDK`는 프로젝트에 추가했으나, `Gamium Engine`를 실행하지 않으려면 **`USE_GAMIUM`**을 해제 해야합니다.
:::

## 3단계 : 코드 추가

아래와 같이 코드를 추가하여 [ServerBuilder](server-configuration)를 통해 `Gamium Engine`를 실행할 수 있습니다.
[ServerBuilder](server-configuration)는 인풋 매핑, 이벤트핸들러 등록 등의 기능을 제공하며 관련내용은 [문서](server-configuration)를 참고부탁드립니다.

```csharp
using System.Collections.Generic;
using Gamium;
using UnityEngine;

public class GamiumServerStarter
{
    [RuntimeInitializeOnLoadMethod(RuntimeInitializeLoadType.BeforeSceneLoad)]
    private static void OnLoad()
    {
        new ServerBuilder().Run();
    }
}
```

## 5단계 : 빌드

[빌드 설정](https://docs.unity3d.com/Manual/BuildSettings.html) 을 확인하여, unity 빌드 후 실행파일을 생성하세요.
해당 실행파일을 가지고 테스트 자동화를 진행할 수 있습니다.
