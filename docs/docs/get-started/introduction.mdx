## About Dogu

Dogu는 게임 개발에 있어서 반복적인 테스트를 자동화하고, 테스트 결과를 시각화하여 개임 개발 및 테스트를 더욱 효율적으로 할 수 있도록 도와주는 툴입니다.

## Features

Dogu는 크게 2가지 서비스를 제공합니다.

- 디바이스 팜
- 게임 테스트 자동화

### 디바이스 팜

Dogu 콘솔에서 조직의 디바이스를 관리할 수 있습니다. 조직 간 디바이스를 자유롭게 공유할 수 있고, 때로는 디바이스를 한 게임 프로젝트에서만 사용하도록 제한할 수도 있습니다.
디바이스가 꼭 모바일 기기일 필요는 없습니다. 데스크탑 기기도 Dogu에서 관리할 수 있습니다.

디바이스 팜을 통해 디바이스와 물리적으로 떨어져 있어도 Dogu 콘솔에서 원격으로 디바이스를 관리할 수 있습니다. 디바이스의 상태를 실시간으로 확인하고, 디바이스의 화면을 실시간으로 스트리밍 하거나, 디바이스를 제어할 수 있습니다.

또한 디바이스를 그룹으로 묶고, 특정 그룹에 속한 디바이스들에 대해 일괄적으로 테스트를 수행하도록 할 수 있습니다.

### 게임 테스트 자동화

Dogu는 게임 테스트를 자동화하기 위해 **Gamium SDK**를 제공합니다.

Gamium SDK는 게임 개발자가 게임 테스트를 자동화하기 위해 Dogu와 통신할 수 있도록 도와주는 라이브러리입니다. Gamium SDK는 Dogu와 통신하기 위한 API를 제공하고, 테스트 결과를 Dogu에게 전달하기 위한 API를 제공합니다.

그리고 작성된 게임 테스트 스크립트들을 디바이스 그룹과 엮어 수행할 수 있습니다.

테스트 결과로 제공하는 디바이스 화면 녹화 영상, 디바이스 프로피일링 정보, 게임 FPS 정보, 게임 로그 등을 Dogu 콘솔에서 확인할 수 있습니다.
