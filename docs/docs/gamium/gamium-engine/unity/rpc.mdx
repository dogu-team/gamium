Gamium은 `Gamium Client`를 통해 Game에 포함된 함수를 원격에서 호출할 수 있는 기능을 제공합니다.

아래와 같은 C#코드는 `Gamium Client`의 [executeRpc](../../gamium-client/api/classes/gamium-client#executeRpc-) 함수를 통해 호출할 수 있습니다.

##### C#

```csharp
internal static class CodebaseSample
{
    public static void CallParam1(double param1)
    {
        Logger.Verbose($"CodebaseSample.CallParam1: {param1}");
    }
}
```

##### Typescript

```ts
await game.executeRpc(
  RpcBy.method('Gamium.Private.CodebaseSample', 'CallParam1', 10),
);
```

### 제약사항

- static 함수만 호출가능
- int, float, string 과 같은 타입만 인자로 전달하고 반환값으로 사용할 수 있습니다
